--How to copy and run things on the server

Step 1) Make runnable jar file
	
	link: https://www.youtube.com/watch?v=qDTUYkaXAEc&t=32s 

Step 2) cd to place with jar file  

Step 3) scp name.jar nickisu@coms-309-sb-4.misc.iastate.edu:name.jar 

Step 4) scp ./name.jar nickisu@coms-309-sb-4.misc.iastate.edu:~

Step 5) go to server

Step 6) java -jar name.jar &
